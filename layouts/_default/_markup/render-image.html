{{- $src := .Destination -}} {{- $alt := .Text -}} {{- $title := .Title -}} {{- $isRemote := or (hasPrefix $src "http://") (hasPrefix $src "https://") -}} {{- $isSpecial := or (hasPrefix $src "#") (hasPrefix $src "data:") -}} {{- if or $isRemote $isSpecial -}} {{- else -}} {{- $cleanSrc := strings.TrimPrefix "./" $src -}} {{- $image := .Page.Resources.GetMatch $cleanSrc -}} {{- if and $image (eq $image.ResourceType "image") (in (slice "jpeg" "jpg" "png" "webp") $image.MediaType.SubType) -}} {{- if eq (.Page.Params.image_processing | default "") "original" -}} {{- else if and (eq $image.MediaType.SubType "png") (ne (.Page.Params.image_processing | default "") "webp") -}} {{- $webpSrc := replaceRE "\\.[^.]+$" ".webp" $cleanSrc -}} {{- $webp := .Page.Resources.GetMatch $webpSrc -}} {{- if $webp -}} {{- else -}} {{- end -}} {{- else if eq $image.MediaType.SubType "webp" -}} {{- else -}} {{- $maxWidth := .Page.Params.image_max_width | default 1200 -}} {{- $srcsetWidths := .Page.Params.image_srcset_widths | default (slice 480 768 1200) -}} {{- $webpOptions := .Page.Params.image_webp_options | default "q95" -}} {{- $fallbackWidth := cond (gt $image.Width $maxWidth) $maxWidth $image.Width -}} {{- $fallback := $image.Resize (printf "%dx webp %s" $fallbackWidth $webpOptions) -}} {{- $srcset := slice -}} {{- range $srcsetWidths -}} {{- $width := int . -}} {{- if le $width $image.Width -}} {{- $resized := $image.Resize (printf "%dx webp %s" $width $webpOptions) -}} {{- $srcset = $srcset | append (printf "%s %dw" $resized.RelPermalink $width) -}} {{- end -}} {{- end -}} {{- end -}} {{- else -}} {{- end -}} {{- end -}}
