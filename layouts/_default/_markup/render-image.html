{{- $src := .Destination -}} {{- $alt := .Text -}} {{- $title := .Title -}} {{- $isRemote := or (hasPrefix $src "http://") (hasPrefix $src "https://") -}} {{- $isSpecial := or (hasPrefix $src "#") (hasPrefix $src "data:") -}} {{- if or $isRemote $isSpecial -}} {{- else -}} {{- $cleanSrc := strings.TrimPrefix "./" $src -}} {{- $image := .Page.Resources.GetMatch $cleanSrc -}} {{- if and $image (eq $image.ResourceType "image") (in (slice "jpeg" "jpg" "png" "webp") $image.MediaType.SubType) -}} {{- if eq $image.MediaType.SubType "png" -}} {{- $webpSrc := replaceRE "\\.[^.]+$" ".webp" $cleanSrc -}} {{- $webp := .Page.Resources.GetMatch $webpSrc -}} {{- if $webp -}} {{- else -}} {{- end -}} {{- else if eq $image.MediaType.SubType "webp" -}} {{- else -}} {{- $maxWidth := 1200 -}} {{- $fallbackWidth := cond (gt $image.Width $maxWidth) $maxWidth $image.Width -}} {{- $fallback := $image.Resize (printf "%dx webp q95" $fallbackWidth) -}} {{- $srcset := slice -}} {{- range slice 480 768 1200 -}} {{- if le . $image.Width -}} {{- $resized := $image.Resize (printf "%dx webp q95" .) -}} {{- $srcset = $srcset | append (printf "%s %dw" $resized.RelPermalink .) -}} {{- end -}} {{- end -}} {{- end -}} {{- else -}} {{- end -}} {{- end -}}
