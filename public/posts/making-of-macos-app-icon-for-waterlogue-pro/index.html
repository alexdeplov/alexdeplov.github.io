Waterlogue Pro MacOS app icon creation

Jun 24, 2024

Project goal

A friend of mine and a client — Tinrocket —asked to make a new Mac app icon for Waterlogue Pro.

Back days iOS version of the icon was used for macOS, but John (the founder of Tinrocket) wanted to make it to be less childish and more “Pro”.

How

As a main designer on that project I started with hand-drawn sketches to catch a vibe of a new icon.

Hand-drawn sketches are a great way of searching for different shape variants quickly. The shape should be simple, bold, and visually interesting.

For that project I used my favorite Winsor & Newton colored pencils, with very bright and saturated colors. It was the perfect choice to draw water splash and translucency effects.

I showed variants to John and he picked version with whale tail pointing up. It wasn’t my the most favourite variant during the time, since other icon had more dynamic shape. But John has great sense of taste, as been shown many times before, so I agreed to proceed with this one.

Process

I decided to start the work with vector waves shapes first, to use it as a foundation for the whale tail. I made few versions of waves, and we discussed it with John again. He’s comment was to fix the tail corners from squared o more smoothed. As we can see later it made a big, positive difference.

I used circular golden ratio grid to make sure that waves and the tail fits perfectly in a balanced composition.

Proportions of the tail was calculated based on a wave wavelength. As you can see here front wave peaks used as an alignment for other elements to build up the relationship between them.

During next sync John pointed out that even if last wave peak was hidden under the icon cut, we still can shift it, to make the icon visually more balanced. As well as to make tail base curve to be a circle, to make it more smooth and friendly.

Results

Final version of the icon was tested with different colors, in order to find a new look. But at the end was decided to re-color it with original Waterlogue icons to keep the brand.

The project was done in about two months. Adobe Illustrator was used as the main tool.

Waterlogue Pro was launched on Mac App Store and after some time become even more successful than iOS version of the app.

Learnings

During this project I learn that client’s vision plays significant role, since the client knows its product better, as well as his audience and the brand that was established years before.

I’m very proud of the results, since the icon appears to be memorable, unique, with friendly smoothed and balanced shapes. The goal of make it to be more Pro, as opposite to childish was achieved.
